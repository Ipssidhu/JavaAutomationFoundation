- Features: Add patients, prevent duplicates, validate age, search by patient ID
- Tech: Java, OOP (Encapsulation, Inheritance, Polymorphism), Collections (List, Set, Map), Custom Exceptions
- Design:
  - `Patient` ‚Üí model entity
  - `PatientService` ‚Üí business logic layer (add/search/validate)
  - `DuplicatePatientException`, `InvalidAgeException` ‚Üí validation errors
  - `ClinicalAppTest` ‚Üí runner simulating end-to-end usage
